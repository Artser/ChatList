# ChatList

Минимальное приложение на Python с графическим интерфейсом на PyQt5.

## Установка

Установите зависимости:

```powershell
pip install -r requirements.txt
```

## Запуск

Запустите приложение:

```powershell
python main.py
```

## Сборка исполняемого файла

Для создания исполняемого .exe файла используйте скрипт сборки:

```powershell
.\build.ps1
```

Или вручную с помощью PyInstaller:

```powershell
pyinstaller --onefile --windowed --name "ChatList" main.py
```

Исполняемый файл будет находиться в папке `dist\ChatList.exe`.

### Параметры сборки:
- `--onefile` - создает один исполняемый файл
- `--windowed` - скрывает консольное окно (для GUI приложений)
- `--name "ChatList"` - имя выходного файла

## Описание

Приложение создает простое окно с меткой и кнопкой, демонстрирующее базовую функциональность PyQt5.
